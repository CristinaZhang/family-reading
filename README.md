## 一、需求
打造一个**轻量化、家庭共享、简单好用**的阅读管理工具，满足家庭日常阅读记录、统计、打卡需求。

### 核心需求
1. **书籍管理**
   - 添加书籍：书名、作者、分类、封面
   - 标记状态：未读 / 阅读中 / 已读完
2. **阅读打卡**
   - 记录每日阅读时长/页数
   - 支持家庭成员区分打卡
3. **阅读统计**
   - 个人阅读时长统计
   - 家庭总阅读统计
   - 按日/周/月查看趋势
4. **界面要求**
   - 简洁、干净、易操作
   - 支持电脑/手机访问
   - 无需复杂登录，家庭内部使用

### 目标用户
家庭成员（父母+孩子），无需技术背景即可使用。
